## 🆘 Troubleshooting

### "API Key Missing"
```
Solution: Set OPENAI_API_KEY in one of:
- Environment variable
- .env file  
- .streamlit/secrets.toml
```

### "No flashcards generated"
```
Solution: Ensure PDF processed successfully
- Check Summary tab for content
- Try another PDF
- Increase max_tokens
```

### "Audio won't play"
```
Solution: Summary must be generated first
- Click "Process PDF" completely
- Wait for "Summary generated" message
- Then try Audio tab
```

### "Progress dashboard is empty"
```
Solution: Take at least one quiz first
- Go to Quiz or Exam tab
- Take a quiz/exam
- Progress will appear in Tab 9
```
