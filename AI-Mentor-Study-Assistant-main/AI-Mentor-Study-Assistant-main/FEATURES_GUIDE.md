## 📌 TIER 1: CORE ENHANCED FEATURES

### 🎴 **Flashcard Generator**
**Location:** Tab 5 - "🎴 Flashcards"

**What it does:**
- Automatically generates flashcards from your PDF notes
- Create 5-30 flashcards per study session
- Interactive flip-card UI (front → back)
- Navigate between cards with Previous/Next buttons

**How to use:**
1. Upload PDF and click "Process PDF"
2. Go to Flashcards tab
3. Set number of flashcards (5-30)
4. Click "📚 Generate Flashcards"
5. Use Previous/Next to navigate
6. Click "🔄 Flip Card" to reveal answers
7. Download in **Anki format** for use with Anki, Quizlet, or other apps

**Export Options:**
- **Anki Format (.txt)**: Compatible with Anki, Memorang, other spaced-repetition apps
- Tab-separated format: Question | Answer

## ⚠️ TROUBLESHOOTING

### Issue: "OpenAI API Key Missing"
**Solution:** Set your OPENAI_API_KEY in:
- Environment variable: `set OPENAI_API_KEY=sk-...`
- `.env` file: `OPENAI_API_KEY=sk-...`
- Streamlit secrets: `.streamlit/secrets.toml`

### Issue: Audio not generating
**Solution:** Summary must be generated first via "Process PDF"

### Issue: Flashcards showing blank
**Solution:** Ensure PDF extraction worked. Try another PDF.

### Issue: Progress not showing
**Solution:** Take at least one quiz first to see progress data
